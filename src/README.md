# GraphQL Benchmarker

This repository contains a naive GraphQL server implementation and seven variations of the implementation:

* [Naive approach](https://github.com/LiUGraphQL/LinGBM-OptimizationTechniquesExperiments/blob/master/src/naive)
* [Caching approach](https://github.com/LiUGraphQL/LinGBM-OptimizationTechniquesExperiments/blob/master/src/cache)
* [Batching-(caching) approach](https://github.com/LiUGraphQL/LinGBM-OptimizationTechniquesExperiments/blob/master/src/batch-cache)
* [Field-Specific approach](https://github.com/LiUGraphQL/LinGBM-OptimizationTechniquesExperiments/blob/master/src/field-specific)
* [Field-Specific with caching approach](https://github.com/LiUGraphQL/LinGBM-OptimizationTechniquesExperiments/blob/master/src/field-specific-cache) 
* [Field-Specific with batching-(caching) approach](https://github.com/LiUGraphQL/LinGBM-OptimizationTechniquesExperiments/blob/master/src/batch-cache-fieldSpecific)
